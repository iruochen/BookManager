# BookManager
## 技术栈
- 后端技术：Spring + SpringMVC + MyBatis（SSM）
- 前端技术：LayUI
- 数据连接池：Druid
- 数据库：MySQL
- 项目管理：Maven
- 版本控制：Git
- Web容器：tomcat
- 反代服务器：nginx
- 云产品：腾讯云COS

## 项目介绍
- 基于 SSM+LAYUINIMI 的教材管理系统

## 主要功能
- 本系统主要包括三种角色
- 管理员
    - 教材信息管理
    - 教材采购功能
    - 教师信息管理
    - 学生信息管理
- 教师
    - 教材申请功能
- 学生
    - 教材领取功能
